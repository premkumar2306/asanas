- Node.js & npm installed
- Firebase CLI (for deployment)
- Git (to clone the repo)

### Installation

```bash
git clone https://github.com/your-username/asanas.git
cd asanas
npm install
npm run dev
```

To build for production:
```bash
npm run build
```

To deploy (if using Firebase):
```bash
firebase deploy
```
